Pescapp

Plataforma de monitoreo de pesca artesanal

Descripción y Objetivo

Pescapp es una plataforma integral desarrollada para el monitoreo eficiente de actividades de pesca artesanal, permitiendo recopilar datos geoespaciales y estadísticos promoviendo la trazabilidad de productos pesquero y la seguridad en el mar de las personas que se dedican a la pesca. El sistema se compone de una aplicación móvil para la recolección de datos en campo y un tablero digital administrativo para su análisis y visualización.

El objetivo principal es proporcionar una herramienta tecnológica que facilite el seguimiento de las actividades pesqueras, contribuyendo a la gestión sostenible de los recursos marinos y a la toma de decisiones basadas en datos precisos. Este proyecto es fruto de la colaboración entre la Universidad Autónoma de Baja California (UABC) y el Colegio de la Frontera Sur (ECOSUR).

Componentes del Proyecto

Aplicación Móvil

Herramienta de campo para pescadores y técnicos que permite registrar rutas, coordenadas, tiempos y otros datos relevantes durante las actividades de pesca.

Funciona con y sin conexión a internet, sincronizando los datos cuando se restablece la conectividad.

Descargar App

Dashboard Administrativo

Plataforma web de análisis desarrollada con Streamlit que permite visualizar rutas de pesca, generar estadísticas y exportar informes.

Incluye diferentes niveles de acceso: administrador, monitor y usuario, permitiendo un control adecuado de la información.
